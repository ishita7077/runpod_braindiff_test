Read · Seven cortical dimensions

What your result is going to show you.

Every comparison decomposes into seven brain systems. Each answers a different question. Scan this while you wait — it makes the result map easier to read.

01 · Personal relevance

Does it feel like it's about me?

Medial prefrontal cortex

How the brain answers "is this about me?" Versions that hit personally light this up harder — often without the reader knowing why.

02 · Social reasoning

Am I modeling another mind?

Temporoparietal junction

Lights up when one mind models another. If a version asks the reader to consider someone else's motives, intent, or feelings, this region carries that signal.

03 · Processing effort

How hard is the system working?

Dorsolateral prefrontal cortex

Effort isn't automatically good or bad. Sometimes a challenging message earns attention; sometimes it loses it. The map lets you see which is happening.

04 · Language depth

How deeply is meaning being parsed?

Broca's and Wernicke's networks

The brain's canonical language systems. Big shifts here usually mean one version is structurally richer — more syntactic or semantic texture — not just longer.

05 · Gut reaction

Does this hit viscerally?

Anterior insula

Tracks felt salience — the bodily edge of a message. If one version provokes a stronger gut response, the insula shows it, even when the literal content is nearly identical.

06 · Memory encoding

Will this stick?

Left ventrolateral prefrontal cortex

A cortical driver of memory formation. Two versions can feel similar in the moment but differ in what endures. If you care about a message lasting, watch this dimension closely.

07 · Attention

Is the system orienting toward this?

Dorsal attention network

Reflects what the reader selects and holds focus on. Content that commands attention through novelty, urgency, or contrast pulls this network harder than content that drifts.
