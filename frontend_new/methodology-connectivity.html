Methodology · Connectivity

The connectivity map.

Each cortical system has a per-second activation timeseries from TRIBE v2. The connectivity map asks a simple question: as the audience moves through the run, which systems engage together, and which engage independently?

Honest scope. We compute Pearson correlation on the predicted per-system activations from TRIBE v2, not on raw fMRI from your audience. Predicted timeseries inherit the structure of the model's encoder and the population it was trained on, so this is a population-average signal — informative for talking about how content engages cortical systems together, but not a direct measurement of any individual viewer's brain. It also does not capture sub-second dynamics: TRIBE works at roughly one prediction per second.

The algorithm

For each pair of cortical systems (21 pairs across 7 systems), we compute the Pearson correlation between their per-second activation timeseries over the full run. The 7×7 symmetric matrix becomes a graph: nodes are systems, edges are correlations.

for each pair (i, j) in the 7 cortical systems: r = Pearson(timeseries_i, timeseries_j) matrix[i][j] = r if abs(r) >= 0.30: draw an edge between system i and system j edge thickness = abs(r) edge color = green if r > 0 (engaged together) = vermillion if r < 0 (engaged in opposition) integration_score = mean of abs(off-diagonal correlations) parallel_score = mean of negative correlations hub_node = the system with the highest sum of |r| across edges isolated_node = the system with the lowest

What integration tells you. A high integration score (≥0.45) means the seven systems engage together — when attention rises, memory tends to rise with it; when language depth rises, social thinking does too. Tightly-coupled cortical state. A low integration score (<0.25) means the systems engage independently — emotion fires when reasoning is quiet, or vice versa. Parallel processing.

What the parallel score tells you. The mean of the negative correlations specifically. A more-negative parallel score means more systems are engaging in opposition during the run.

What we don't claim

  • This is not raw functional connectivity. We derive it from TRIBE v2's predicted activations, which are population-average and per-second.
  • It does not predict any individual viewer's behaviour. It describes the structure of the cortical state that a typical cortex would enter while consuming this content.
  • Correlation is symmetric and undirected — it cannot tell you which system is driving which. Granger causality would; we don't compute that here because TRIBE's per-second resolution isn't fine enough to support it cleanly.
  • Edges below |r| = 0.30 are not drawn. The full matrix is always available in the API response if you want to inspect every value.

Citations

  • Power, Cohen, Nelson, Wig, Barnes, Church, Vogel, Laumann, Miezin, Schlaggar, Petersen (2011), Neuron. “Functional network organization of the human brain.” scholar
    Foundation paper for measuring functional integration and segregation via cross-region correlations. Established the dorsal attention, frontoparietal control, default-mode and other canonical networks we name in our results.
  • Sporns (2013), Current Opinion in Neurobiology. “Network attributes for segregation and integration in the human brain.” scholar
    Defines integration (positive cross-network correlations) and segregation (within-network coupling) as the two-axis description of cortical state. Our integration_score and parallel_score sit on this same axis.

Inspect the source
