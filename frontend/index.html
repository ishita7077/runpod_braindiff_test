Methodology

How it works (short)

  • Inputs — paste A/B; we run both through TRIBEv2 on a speech-aligned path.
  • Contrast — signed vertex- and region-level B−A differences on fsaverage5.
  • Output — heatmaps, dimensions, plain-language takeaways with citations.
