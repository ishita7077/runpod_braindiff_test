How Brain Diff Works

Brain Diff compares two versions of text and shows where the average brain model predicts different cortical response. It is a comparison tool, not an absolute score.

1) What this does (in two lines)

You paste Version A and Version B. We run both through TRIBEv2 and show which cortical systems move up or down between the two versions.

2) Where the brain data comes from

TRIBEv2 (Meta FAIR) is trained on 1,115 hours of fMRI recordings from 700+ people. Those scans measure blood-flow changes that track neural activity over time.

3) Why text is converted to speech first

The serving path converts text to speech timing before neural prediction, so outputs reflect an audio-aligned processing pathway. This is intentional and matches how the model is exercised in this product.

4) What the 20,484 numbers are

For each time step, TRIBEv2 predicts activation at 20,484 cortical surface vertices. Think of these as a dense map of tiny points across the cortical sheet.

5) Why these dimensions

We group vertices into interpretable cortical systems: Personal Resonance, Social Thinking, Brain Effort, Language Depth, Gut Reaction, Memory Encoding, and Attention/Salience.

Attention/Salience — Dorsal attention network (intraparietal sulcus + frontal eye fields)
HCP areas: LIPv, LIPd, AIP, VIP, MIP, FEF, PEF (both hemispheres).

This network controls where the brain allocates processing resources. When content commands attention through urgency, novelty, or salience, this region fires more strongly. It sits at the top of the attention → effort → value → memory pipeline.

Corbetta & Shulman, 2002; Kastner & Ungerleider, 2000

6) Atlas + scoring method

Region definitions use HCP MMP1.0 labels. For each region we compute signed mean activation and compare B minus A. Positive means B is higher; negative means A is higher.

7) Brain Effort uses three cognitive-load types

High effort can mean intrinsic load (hard topic), extraneous load (confusing writing), or germane load (productive learning). The signal alone cannot fully disambiguate which one dominates.

8) Memory encoding is a cortical driver, not full memory completion

Memory encoding — a partial signal

Our Memory Encoding dimension tracks left vlPFC activity associated with the Dm effect. Full memory formation also depends on hippocampal circuitry, which this cortical model cannot directly observe.

So treat this as encoding likelihood/drive, not guaranteed long-term recall.

9) What this is not

Not a personal brain readout. Not an engagement or virality guarantee. Not a medical or diagnostic system.

10) Whose "average brain" this reflects

The model reflects aggregate research cohorts, often WEIRD-leaning populations. Cultural, demographic, and context differences can shift how real audiences respond.

Predictions for the population-average brain (TRIBEv2, Meta FAIR). Text processed as speech.
